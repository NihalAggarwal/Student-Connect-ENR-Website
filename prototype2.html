Interest Based Searching

Implementing the Elastic Search feature to enable Interest Based Search We worked on integrating this feature as this is a complete game changer. This enables you to search for people in your College/ University Based on Interests. You can connect with like minded people, get their guidance and get mentored by them.

Our Main Features

Elastic Search for Interest based search

Search for people that share common interests and are working in the domain form your University

Authentication and Community Exclusivity

Complete Privacy and Protection with Access to Organization members with stringent Authorization methods

The process of Integrating the Elastic Search features and Technology that drives it

Elastic search is search engine that provides full text searching feature for an application. We have implemented Elastic Search to facilitate Interest Based Searching and get Efficient Search Results according to the the Search of Interest. This has been implemented on the Firebase base console which makes the proess hassle free.

The process of Integrating the Authorization features and Technology that drives it

The Authorization process is very vital to our prototype as it aims at making the exclusive for a College Community.This feature provides us scalability as we can now tweak and make it authorizable to other users colleges on demand. We have implenented this with the help of Auth Methods from Cloud Firestore.
